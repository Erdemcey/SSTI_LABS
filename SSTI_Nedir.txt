

--- Server Side Template Injection ----


    Bu zafiyeti anlamak için önce template kavramını anlamalıyız. Template giderek artan
web ihtiyaçlarında artan girdi tiplerini sınıflandırmak , sınıflandırırkende karmaşık
kod yazımından kaçınmak için kullanılır. Temelinde kodlama dilidir.


Template örnekleri:

-- PHP – Smarty, Twig
-- Java – Velocity, Freemaker
-- Python – Jinja2, Mako, Tornado
-- JavaScript – Jade, Rage
-- Ruby – Liquid


    Bu zafiyetler tıpkı xss gibi işler, zafiyeti tespit etmek için {{7*7}} gibi bir payload
yazarak tespit etmeye çalışırız. Dönen sonuç 49 ise Twig 7777777 ise jinja2 şablonu kullanılıyor demektir.
Bu noktada bir çok zafşyet tespiti stratejisi var,sadece bunlar değil yani.


    Zafiyet taspit edildikten sonra sömürü kısmına geçiyoruz. Sömürü için ise bu şablonların dökümanını 
okuyup kendi payloadımızla RCE ye kadar uzanacak bir saldırı başlatabiliriz.

Örnek payloadlar:

    Python2

    {{ ''.__class__.__mro__[2].__subclasses__()[40]('/etc/passwd').read() }}
    

    Python3

    {{ ''.__class__.__mro__[1].__subclasses__()[40]('/etc/passwd').read() }}
